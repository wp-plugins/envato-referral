== Description ==

A simple plugin that creates shortcodes and widgets for inserting referrals to envato.com sites

The shortcodes are:

	envato_referral_link
	envato_referral_list

The widgets are:

	EnvatoReferralLink
	EnvatoReferralList

Shortcode / Widget Options:

	OPTION            DEFAULT
	--------------------------------
	user_name         blasher
	envato_site       codecanyon.net
	image_version     v1
	image_size        125x125
	max_width         535px


Major new features in EnvatoReferral 0.4.002 include:

-jquery cycle effects for image_version=cycle



== Installation ==

1) Upload the plugin to your blog, 2) Activate it, then 3) Modify the settings appropriately and add shortcodes .

1, 2, 3: You're done!
