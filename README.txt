== Description ==

A simple plugin that creates shortcodes and widgets for inserting referrals to envato.com sites

Major new features in EnvatoReferral 0.0.2 include:

* A comment status history, so you can easily see which comments were caught or cleared by Akismet, and which were spammed or unspammed by a moderator




== Installation ==

1) Upload the plugin to your blog, 2) Activate it, then 3) Modify the settings appropriately.

1, 2, 3: You're done!
